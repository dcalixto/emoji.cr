# emoji.cr

Emoji library for Crystal. Inspired by [Emoji for Python](https://github.com/carpedm20/emoji)

**Work in progress.**

## Installation

Add it to `Projectfile`

```crystal
deps do
  github "veelenga/emoji.cr"
end
```

## Usage

```crystal
require "emoji.cr"
```
